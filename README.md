# 🎓 College Placement Portal

A centralized platform designed to bridge the gap between students, the college placement cell, and recruiters. This application streamlines the campus recruitment process, making it easier to manage job postings, student applications, and placement tracking.

## 🚀 Tech Stack

**Frontend:**
* React.js (Vite)
* Tailwind CSS / UI Components

**Backend:**
* Java Spring Boot
* Spring Security (Authentication & Authorization)

**Infrastructure & DevOps:**
* **Cloud:** AWS (VPC, EC2, Private/Public Subnets)
* **Web Server / Reverse Proxy:** Nginx
* **CI/CD:** GitHub Actions (Automated Build & Deployment)
